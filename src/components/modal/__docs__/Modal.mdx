# Modal

Modal presents content in a focus-trapped glass surface over an application.

#### Example

<Canvas of={ModalStories.Default} />

## Usage

```ts
import { Modal, Button } from "react-glass-ui";
import { useState } from "react";

const Example = () => {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button text="Open modal" onClick={() => setOpen(true)} />
      <Modal
        open={open}
        onOpenChange={setOpen}
        title="Glass modal"
        description="A translucent surface that keeps the focus on your content."
        footer={
          <>
            <Button text="Cancel" variant="link" onClick={() => setOpen(false)} />
            <Button text="Confirm" onClick={() => setOpen(false)} />
          </>
        }
      >
        <p>
          This modal demonstrates the glassmorphism style with focus management, overlay dismissal,
          and a flexible footer.
        </p>
      </Modal>
    </>
  );
};

export default Example;
```

#### Props

- **open** _`boolean`_ — Controls whether the modal is visible.
- **onOpenChange** _`(open: boolean) => void`_ — Callback fired when the open state should change.
- **onClose** _`() => void`_ — Fired after the modal requests to close (overlay, close button, or escape).
- **title** _`React.ReactNode`_ — Heading content displayed at the top of the modal.
- **description** _`React.ReactNode`_ — Supporting text rendered under the title.
- **footer** _`React.ReactNode`_ — Footer content, typically actions.
- **size** _`"sm" | "md" | "lg"`_ — Controls the maximum width. Default is `"md"`.
- **closeOnOverlay** _`boolean`_ — Allows closing by clicking the overlay. Default is `true`.
- **closeOnEsc** _`boolean`_ — Allows closing with the Escape key. Default is `true`.
- **lockScroll** _`boolean`_ — Prevents body scroll while open. Default is `true`.
- **enableLiquidAnimation** _`boolean`_ — Enable liquid bounce animation on modal open. Default is `true`.
- **portalContainer** _`HTMLElement | null`_ — Custom container for rendering the modal.
- **className** _`string`_ — Additional CSS classes to apply to the modal content.
