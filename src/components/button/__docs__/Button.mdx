# Button

Button component with different props.

#### Example

<Canvas of={Button.Default} />

## Usage

```ts
import { Button } from "react-glass-ui";

const Example = () => {
  return (
      <Button
        onClick={()=> console.log("Clicked")}
        variant={"default"}
        size={"medium"}
      >
        Click me!
      </Button>
  );
};


export default Example;
```

#### Arguments

- **text** _`string`_ - A string that represents the text content of the button.
- **children** _`React.ReactNode`_ - Content to render inside the button. Takes priority over `text` prop if both are provided.
- **variant** _`"default" | "positive" | "negative" | "warning"`_ - The visual style variant of the button. Uses colors from tailwind.config.
- **disabled** _`boolean`_ - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **size** _`"small" | "medium" | "large"`_ - Defines the size or dimensions of the button. Default is `"medium"`.
- **rounded** _`boolean`_ - Makes the button circular. When true, the button will have a 1:1 aspect ratio with rounded corners. Default is `false`.
- **onClick** _`(event: React.MouseEvent<HTMLButtonElement>) => void`_ - A function that is called when the button is clicked.
- **enableClickAnimation** _`boolean`_ - A boolean indicating whether to enable the liquid bounce animation on click. Default is `true`.
- **className** _`string`_ - Additional CSS classes to apply to the button.
